Sobre PWA

Uma aplicação PWA deve estar disponível para diferentes tipos de usuários, tanto em computadores quanto em smartphones, funcionando de forma responsiva e acessível.

Uma das principais características de um PWA é a possibilidade de funcionamento offline. Isso acontece porque alguns arquivos são armazenados em cache pelo navegador.

Quando o usuário acessa o site pela primeira vez, os arquivos principais são salvos. Depois disso, mesmo sem internet, o aplicativo pode continuar carregando as páginas armazenadas.

Além disso, o PWA pode ser instalado no celular, criando um ícone na tela inicial, como se fosse um aplicativo comum.
