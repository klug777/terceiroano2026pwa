Progressive Web App - Atividade Prática

Este site foi desenvolvido utilizando HTML, CSS e pode ser instalado como um aplicativo PWA.

Por que usar PWA?

  • Funciona offline.
  • Pode ser instalado no celular.
  • Tem desempenho rápido.
  • Permite recursos modernos, como cache e notificações.

Aplicações Progressivas

PWA significa Progressive Web App, ou Aplicação Web Progressiva. Esse tipo de aplicação busca funcionar em diferentes dispositivos, com boa experiência para o usuário e recursos parecidos com os de um aplicativo nativo.
